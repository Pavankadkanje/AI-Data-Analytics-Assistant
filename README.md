# 📊 AI Data Analytics Assistant

An interactive AI-powered Data Analytics application built using **Python**, **Streamlit**, **Pandas**, **Plotly**, and **Scikit-learn**.

The application enables users to upload datasets, clean data, visualize insights, generate reports, perform basic machine learning predictions, and interact with the dataset through an AI-style chat interface.

---

## 🚀 Features

- 📁 Upload CSV and Excel datasets
- 📊 Dataset Profiling
- 🧹 Data Cleaning
  - Remove Duplicates
  - Handle Missing Values
  - Remove Empty Rows
- 📈 Interactive Visualizations
  - Bar Chart
  - Line Chart
  - Scatter Plot
  - Histogram
  - Pie Chart
  - Box Plot
  - Correlation Heatmap
- 🤖 AI Chat for dataset queries
- 📉 Machine Learning Prediction
- 📄 Report Generation
  - CSV Download
  - Excel Download
  - PDF Report

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/yourusername/AI-Data-Analytics-Assistant.git
```

### Move into the project

```bash
cd AI-Data-Analytics-Assistant
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
streamlit run app.py
```
